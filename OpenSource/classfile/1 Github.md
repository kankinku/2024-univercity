# 1. Github

## Github의 기능

1. Source code : 소스코드 저장 및 공유
2. Version control : 소스코드 업데이터 관리

> github → repositories( repo ) : 저장소
→ git을 통해서 저장된 코드를 다운로드 받는다.
> 

> 누국가의 project A
 → pork를 통해서 사용자화 가능하다.
